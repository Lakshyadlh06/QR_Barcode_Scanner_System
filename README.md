# QR_Barcode_Scanner_System

This project is a real-time and image-based QR/Barcode scanner built using Python. It uses OpenCV for image processing and pyzbar for decoding barcodes and QR codes.

The system can:

📡 Scan codes live using a webcam
🖼️ Decode codes from static images
🏷️ Display decoded data directly on the screen
